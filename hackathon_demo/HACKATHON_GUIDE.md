## 🤖 Automated Demo Script

We've provided a standalone Playwright script if you want to run the demo LIVE on the main stage or record a high-fps video.

### How to Run:
1.  **Dependencies**: Ensure the app is running (`npm run dev`).
2.  **Execute**:
    ```bash
    node hackathon_demo/test_modes.js
    ```
    *Note: This script will automatically refresh the vault with fresh high-resolution screenshots.*
